======================
Petunjuk Pemasangan
======================
1. Cari file elerning.sql yang ada pada folder config
2. Buat Database denga nama 'elerning' lalu Importkan file elerning.sql ke database tersebut (phpmyadmin)
3. Lakukan Configurasi Database (username, password, serveer/host sesuai dengan phpmyadmin anda) 
pada file config.php yang terdapat pada folder config/config.php
4. Lalu jalankan aplikasi
